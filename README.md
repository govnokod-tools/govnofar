Плагин для Far3@x64 а Бормандовскому API гэкашечки.
Добавляете far в %PATH%, в проекте Govnokod -> properties -> Debugging -> Command ставите far.exe
Жмете Play
Плагин собирается и копируется вам в "%APPDATA%\far manager\Profile\Plugins\Govnokod"
(см deploy.bat).

Работать лучше с Release версией.

Govnolib -- либа-клиент для стат. линковки
Govnotool её консольный клиент.
libson либа для парса json
Govnokod сам плагин
FarSDK -- FarSDK

Если просто хотите взять release:
.dll копировать в ``"%APPDATA%\far manager\Profile\Plugins\Govnokod"`` или
``c:\Program Files\Far Manager\Plugins\Govnokod``

Запускать через ALT+F1 или ALT+2.
Смотреть текст F3 или F4. 
F8 скрывает пользователя, сохраняя в БД "%LOCALAPPDATA%\Far manager\Profile\PluginsData"
